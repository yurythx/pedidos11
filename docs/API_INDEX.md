# Mapa de Endpoints da API

- Índice
  - GET /api/v1/ → lista vendas, estoque, financeiro, relatorios, auditoria
- OpenAPI
  - GET /api/schema/ → schema OpenAPI em JSON
- Vendas
  - GET /api/v1/vendas/produtos/
  - GET /api/v1/vendas/produtos/{slug}/
  - GET /api/v1/vendas/categorias/
  - GET /api/v1/vendas/categorias/{slug}/produtos/
  - GET /api/v1/vendas/pedidos/
  - POST /api/v1/vendas/pedidos/
  - GET /api/v1/vendas/pedidos/{slug}/
- Estoque
  - GET /api/v1/estoque/movimentos/
  - POST /api/v1/estoque/movimentos/entrada
  - POST /api/v1/estoque/movimentos/ajuste
  - GET /api/v1/estoque/movimentos/saldo?produto=<slug>
  - GET /api/v1/estoque/movimentos/historico?produto=<slug>&start=YYYY-MM-DD&end=YYYY-MM-DD
  - GET /api/v1/estoque/recebimentos/
  - POST /api/v1/estoque/recebimentos/
  - GET /api/v1/estoque/recebimentos/csv?fornecedor=<slug>&deposito=<slug>&start=YYYY-MM-DD&end=YYYY-MM-DD
  - GET/POST /api/v1/estoque/depositos/
  - GET/POST /api/v1/estoque/motivos-ajuste/
  - POST /api/v1/estoque/movimentos/transferir
  - POST /api/v1/estoque/recebimentos/{id}/estornar/
- Financeiro
  - GET /api/v1/financeiro/lancamentos/
  - GET /api/v1/financeiro/lancamentos/resumo?... e *_csv
  - GET/POST /api/v1/financeiro/contas
  - GET/POST /api/v1/financeiro/centros
  - GET/POST /api/v1/financeiro/defaults (superuser)
- Relatórios
  - GET /api/v1/relatorios/faturamento e faturamento_csv
  - GET /api/v1/relatorios/itens e itens_csv
  - GET /api/v1/relatorios/categorias e categorias_csv
  - GET /api/v1/relatorios/estoque_depositos e estoque_depositos_csv
  - GET /api/v1/relatorios/ajustes_motivo e ajustes_motivo_csv
- Auditoria
  - GET /api/v1/auditoria/
  - GET /api/v1/auditoria/{id}/
 - Cadastro
  - GET/POST /api/v1/cadastro/fornecedores/
  - GET/POST /api/v1/cadastro/enderecos/
  - GET/POST /api/v1/cadastro/clientes/
