Estoque

Recebimentos
curl -X POST "{{ estoque }}recebimentos/" -H "Content-Type: application/json" -d "{\"fornecedor\":\"fornecedor-x\",\"documento\":\"NF-123\",\"deposito\":\"deposito-central\",\"itens_payload\":[{\"produto\":\"suco\",\"quantidade\":10}]}"
Depósitos
curl "{{ estoque }}depositos/"
Transferência entre depósitos
curl -X POST "{{ estoque }}movimentos/transferir/" -H "Content-Type: application/json" -d "{\"produto\":\"suco\",\"quantidade\":2,\"origem\":\"deposito-central\",\"destino\":\"loja-1\"}"
Estornar recebimento
curl -X POST "{{ estoque }}recebimentos/1/estornar/"
